# CHANGELOG

## 2026.05.08

### What Changed

Bumped kernel version from 7.0.1 (tagrel 2) to 7.0.5 (tagrel 1), matching the latest CachyOS upstream release `cachyos-7.0.5-1`.

### Technical Details

Only `_minor` and `_tagrel` changed in PKGBUILD. The b2sums are now stale and will be regenerated by `updpkgsums` when `./build-kernel.sh` is run. The existing `config` file is forward-compatible — new kernel config options introduced in 7.0.2–7.0.5 are handled automatically by `yes "" | make config` during the build, which accepts kernel defaults for any new symbols not present in the current .config.

### Files Modified

- `PKGBUILD` — `_minor=1` → `_minor=5`, `_tagrel=2` → `_tagrel=1`
- `CLAUDE.md` — updated Current State to reflect new version
